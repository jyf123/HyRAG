# HyRAG
The implementation for "Generalizing Graph Foundation Models via Hyperbolic Retrieval-Augmented Generation" at KDD 2026.

## Dependencies

For install dependencies, run
```
pip install -r requirements.txt
```

## Experiments
We have included the Cora dataset in this project. For more datasets, please download them from: https://github.com/ZhuYun97/GraphCLIP.

To evaluate our proposed method, please execute the following command:

```
 sh run.sh
```
